## 🚀 使用方法

### 环境要求

```bash
Python 3.10+
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

### 学习路径

1. **数学基础** → 阅读 `math/` 下的 Markdown 笔记（线性代数、概率统计已详细更新）
2. **统计学专题** → 学习 `statistics_for_ds/` 数据科学家统计学指南 🆕
3. **Python 编程** → 学习 `python/` 下的示例代码
4. **实践整合** → 运行 `notebooks/` 中的 Jupyter Notebook

### 运行示例

```bash
# 进入仓库
cd DataScience_Math_Python

# 运行 Python 示例
python python/numpy/numpy_basics.py

# 打开 Notebook
jupyter notebook notebooks/
```
