本仓库旨在为数据科学初学者到中级学习者提供：
- 系统的数学知识笔记（含 LaTeX 公式）
- 可运行的 Python 示例代码

## 🚀 使用方法

### 环境要求

```bash
Python 3.10+
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

### 学习路径

1. **数学基础** → 阅读 `math/` 下的 Markdown 笔记
2. **Python 编程** → 学习 `python/` 下的示例代码
3. **实践整合** → 运行 `notebooks/` 中的 Jupyter Notebook

### 运行示例

```bash
# 进入仓库
cd DataScience_Math_Python

# 运行 Python 示例
python python/numpy/numpy_basics.py

# 打开 Notebook
jupyter notebook notebooks/
```
